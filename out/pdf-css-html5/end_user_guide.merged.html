Caculator App

Using a caculator app.

You can use a calculator app on your smartphone. You will find it useful when you need to quickly calculate some numbers.

  1. Navigate to the calculator application icon.
  2. Tap the icon.
  3. The application starts.
You are now ready to calculate with the app.
For example, if you have purchased a book for 5$ and you have given 12$, how much change should you expect? Answer is 18-10=8 $ change.
